Class: Search

Search

Creates an instance of Search. This is the most important class. This class has several data structures and methods used to update the application and control the user interaction. The class has a method "initApp" that initializes all the necessary in order to make the application work. The properties currentSearch, loadDataController, appUI, appFilters and map have their own reset methods to restore their default values.
